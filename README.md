# Ashwin_M_Minor_Project_Automated_YARA_Rule_Generation_RAG
Automated framework for high-precision YARA rule generation using a Retrieval-Augmented Generation (RAG) pipeline. Embeds malware/benign metadata, retrieves positive/negative neighbors with FAISS, uses a Benign-Aware LLM prompt, and applies greedy optimization to produce a compact, high-coverage ruleset.
